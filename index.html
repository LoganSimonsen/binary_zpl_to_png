ZPL Label File to PNG Converter

Convert a `.zpl` label file into a previewable PNG using the Labelary API.

Upload your `.zpl`, select print density and dimensions, then generate or download the PNG.

Why this tool? Unlike labelary.com's web interface, this handles `.zpl` files with embedded binary image data that cannot be pasted as text.
